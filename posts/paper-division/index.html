Paper Division

Explanation of paper divisions—the math to divide N grids

Tags: MATH ORIGAMI

By Obert Guo

Published Saturday, May 22, 2021


Introduction

In origami, there may be times where a 13 x 13 grid or an 11 x 11 grid is needed. Such “indivisble” grids are hard to fold without measuring or cutting strips off, since realistically, the paper can only be accurately bisected.

Method

One way to fold an N x N grid is to use reference points.

Let us plot a square on a Cartesian plane, with the bottom-left corner of the square placed about the origin.

Let x represent a point along the x-axis where it lies on a line that is on the N x N grid. Let z represent a known reference crease or point. A diagonal line will bisect the square.

The equation of the diagonal line from the top-right corner to the bottom-right corner can be represented as $$y=-x+1$$

The equation of the diagonal line from the origin to point z on the square can be represented as $$y=zx$$

To find their point of intersection, both equations will be set to equal each other $$-x+1 = zx$$

Which can rearranged to $$z = \frac{-x+1}{x}$$

If z is a well known reference point that is easily divisible, then z can be represented as a positive rational $$z = \frac{a}{b}\text{ where } a,b \in \mathbb{N} \text{ and } b > a$$

Therefore $$\frac{a}{b} = \frac{-x+1}{x}$$ $$ax = -bx + b$$ $$(a+b)x = b$$ $$x = \frac{b}{a+b}$$

Since x represents a point that lies on a line on an N x N grid, then x can also be expressed as a rational where the denominator represents the Nth division

$$N = a+b$$

Thus, any number chosen for a and b such that b is greater than a, could be used to form an N x N division. However, in the context of origami, the number chosen for b should idealy be a power of 2, since only bisections can be done accurately.

$$z = \frac{a}{2^n} \text{ where } 2^n \text{ is the largest power of 2 less than N }(\because b > a)$$

Summary

To divide a square into a N x N division, $$\text{1. Determine the largest power of 2, assigned as $$b$$}$$ $$\text{2. Subtract $$b$$ from N to determine the integer $$a$$}$$ $$\text{3. Create a reference crease at }\frac{a}{b}$$ $$\text{4. The point of intersection between the line from the bottom left corner to the reference crease } \frac{a}{b}$$ $$\text{and the line from the top-left corner to bottom-right corner, is a point that will lie on the Nth divisions}$$ $$\frac{b}{a+b}$$
